# RefugeX

> Анализатор крафта убежища и аукциона для STALCRAFT:X.

## О проекте

RefugeX — десктопное приложение для анализа крафта, ликвидности и прибыли предметов в системе убежища STALCRAFT:X. Проект использует официальный STALCRAFT API для получения актуальных цен, лотов и информации о предметах.

Программа пишется на:
- Python
- PyQt6
- SQLite

by HqteRave and Baozi
