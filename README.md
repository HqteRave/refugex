# RefugeX

> Анализатор крафта убежища и аукциона для STALCRAFT:X.

## О проекте

RefugeX — десктопное приложение для анализа крафта, ликвидности и прибыли предметов в системе убежища STALCRAFT:X.

Проект использует официальный STALCRAFT API для получения актуальных цен, лотов и информации о предметах.

Основная цель проекта — помочь игрокам:

- анализировать прибыльность крафта
- сравнивать стоимость ресурсов
- отслеживать аукцион
- искать выгодные предметы
- оценивать ликвидность рынка
- упрощать работу с убежищем

---

## Основные возможности

- Анализ прибыли крафта
- Сравнение цен ресурсов и готового предмета
- Работа с аукционом STALCRAFT:X
- Фильтрация выгодных крафтов
- Система ликвидности предметов
- Кэширование данных для быстрой работы
- Асинхронная синхронизация API
- Современный интерфейс на PyQt6

## Интерфейс

Приложение построено на PyQt6 и включает:

- главное окно анализа
- калькулятор крафта
- систему сравнения предметов
- индикаторы ликвидности
- динамические карточки предметов

---

## Особенности архитектуры

RefugeX использует современную async архитектуру:

- asyncio
- SQLite-кэш
- автоматические повторы запросов
- защита от rate limit
- фоновая синхронизация данных
- неблокирующий интерфейс
